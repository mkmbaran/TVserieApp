TV-show reading and retrieving application

If you are seeing this page, it means the application is running

This simple application will read src/main/resources/tvserier.txt one line at a time, query tvmaze.com/api for more information about each show, and save it to a local MySQL database

To begin, press the button below.

Once clicked, the application will read the tvserier.txt file, query tvmaze.com/api, and save them to a local MySQL database

This might take a while

Please keep an eye on the Java console for progress.

PSA

I only had enough time to make the application save the data to MySQL tables, no report.txt gets printed. Please connect to MySQL to see data :)
